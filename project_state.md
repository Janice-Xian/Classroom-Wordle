## 已完成功能

### 词库管理
- [x] 文本框粘贴词条，支持英文逗号 / 中文逗号 / Tab 分隔符
- [x] 自动清洗（零宽字符、全角字母、非字母字符）
- [x] 无效行提示、空词库拦截（< 5 个有效词）
- [x] 自动去重（不区分大小写）
- [x] 词库命名（默认当前日期）、上传者昵称（默认"匿名老师"）
- [x] Supabase 云端持久化，所有设备可见
- [x] 冷启动自动插入"示例词库"（10 个内置单词）
- [x] 词库列表按上传时间倒序，支持删除（含确认弹窗）
- [x] 网络加载失败显示友好提示 + 重试按钮

### 游戏核心
- [x] 4-8 字母词过滤，不足 5 个时拦截
- [x] Fisher-Yates 洗牌 + 不放回抽取（防死锁）
- [x] 每轮 5 个词，长词（7-8 字母）最多 3 个
- [x] 按字母数升序排列（同字母数随机顺序）
- [x] 动态网格：4-6 字母 L+1 行，7-8 字母固定 7 行
- [x] 宽松版 Wordle 颜色算法（不消耗字母）
- [x] 逐格翻转动画（300ms 间隔）
- [x] 动画期间锁定输入
- [x] 输入不足按 Enter 抖动动画
- [x] 提示按钮（点击 / 空格键切换），默认隐藏

### 虚拟键盘
- [x] 三行布局：第二行末 Del，第三行末 Enter
- [x] 按键颜色状态（绿 > 黄 > 灰，不降级）
- [x] 进入下一词时颜色重置
- [x] 防止手机原生键盘弹出（onPointerDown + preventDefault）

### 流程控制
- [x] 5 词连闯，完成后庆祝界面
- [x] "下一个" / Enter 键 1.5 秒延迟防误触
- [x] "再来一局"从剩余词池继续抽取
- [x] 按钮防连击

### 持久化与恢复
- [x] sessionStorage 保存全部游戏状态，刷新可恢复
- [x] 游戏开始后断开 Supabase 依赖（词库本地缓存）
- [x] beforeunload 刷新警告（仅游戏进行中）

### UI / 适配
- [x] 100dvh + 100vh fallback，防手机地址栏遮挡
- [x] 手机端（375px）8 列不横向溢出
- [x] 投影仪可读性（大字体、高对比度颜色）
- [x] 虚拟键盘颜色：未猜测 #d3d6da（浅灰）vs 已排除 #787c7e（深灰），对比明显
- [x] 使用说明弹窗
